### Internal Linking Rules

- **EVERY blog post** links to /ai-builders-club with anchor text "AI Builders Club" (at least once)
- Every Tier 3 page links UP to its parent Tier 2 cluster article
- Every Tier 2 article links UP to its Tier 1 pillar + 3-5 siblings
- Every post includes a "Related Resources" section (auto-generated)
- Every post includes an inline CTA banner mid-article: mint-colored bar → /ai-builders-club
- Every post includes end-of-article newsletter signup (Beehiiv embed)
- n8n auto-injects internal links based on keyword matching rules in Airtable

### Production Pipeline

1. **Keyword Queue:** Airtable base with 1,000+ keywords. Replenish weekly via Ahrefs.
2. **AI Generation:** n8n pulls 10 keywords daily → Claude API with brand voice prompts → drafts to staging.
3. **Human Review:** 100% Tier 1, 50% Tier 2, 10% Tier 3.
4. **Publishing:** WordPress REST API via n8n. Auto-formats, assigns categories, injects internal links.
5. **Indexation:** Auto-submit to Google via IndexNow API through n8n.

## Quality Control

| Content | Review Standard | Who | Reject If |
|---------|----------------|-----|-----------|
| Tier 1 pillar articles | 100% human review + edit | You | Factual errors, weak structure, missing depth |
| Tier 2 cluster articles | 50% spot-check | You or VA | Thin (<1,200 words), repetitive, broken links |
| Tier 3 programmatic | 10% weekly audit | You | Duplicate content, template artifacts, 0 indexation |
| LinkedIn posts | 100% before posting | You | Off-brand tone, weak hook, missing CTA |
| Twitter threads | 100% before scheduling | You | Weak hooks, broken flow |
| Lead magnets | 100% before publishing | You | Low value, poor design, broken downloads |
| Newsletter | 100% before sending | You | Factual errors, broken links, off-brand voice |

### Monthly Audits

- Screaming Frog crawl: thin pages, duplicate titles, broken internal links
- GSC review: high impressions + low CTR pages (fix titles/metas)
- Prune pages with 0 impressions after 60 days
- A/B test LinkedIn hooks: which format drives most "AI Builders Club" comments
- Review auto-DM conversion rate: comments → DMs sent → website visits → emails captured
