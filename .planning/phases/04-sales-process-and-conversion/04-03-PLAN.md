  <files>.planning/phases/04-sales-process-and-conversion/deliverables/pipeline/hubspot-pipeline-setup.md</files>
  <action>
Create a step-by-step HubSpot CRM pipeline configuration guide. This is a setup document that Saksham follows to configure HubSpot -- not application code.

**1. Pipeline: AI Consulting Deals**

6 stages with entry/exit criteria and required properties (follow the structure from 04-RESEARCH.md):

Stage 1: LEAD
- Entry: Contact created from /solutions form, LinkedIn DM, or referral
- Required properties: Contact name, company, source (Website/LinkedIn/Referral), ICP segment (PE/Startup/Enterprise)
- Exit criteria: Pre-call questionnaire sent
- Automation (if Sales Hub Starter): Create task "Send pre-call questionnaire" when deal enters this stage

Stage 2: QUALIFIED
- Entry: Pre-call questionnaire received, ICP confirmed
- Required: Questionnaire responses logged in deal notes, budget range estimated
- Exit criteria: Discovery call scheduled on Calendly

Stage 3: DISCOVERY CALL
- Entry: Call scheduled
- Required: Call date, pre-call research completed (3x3 method notes in deal)
- Exit criteria: Call completed, BANT qualified, COI calculated
- Post-call actions: Log COI amount, update tier selection, create task "Draft proposal within 48 hours"

Stage 4: PROPOSAL SENT
- Entry: Proposal document delivered
- Required: Proposal PDF attached to deal, tier selected, price quoted, review meeting date set
- Exit criteria: Proposal review meeting completed
- CRITICAL: Deal MUST NOT enter this stage without a review meeting booked

Stage 5: NEGOTIATION
- Entry: Prospect engaged with proposal
- Required: Decision timeline confirmed, all stakeholders identified, stall reason (if applicable)
- Exit criteria: Verbal yes or formal decline

Stage 6: CLOSED WON / CLOSED LOST
- Won required: Contract signed, first payment received, kickoff date set
- Lost required: Loss reason logged (dropdown: Lost to Competitor / No Budget / No Decision / Timing / Internal Solution), follow-up date set if timing-related
- Lost action: Add to 90-day re-engagement list if reason is Timing

**2. Custom Deal Properties:**
- ICP Segment: PE Firm / Funded Startup / Enterprise (dropdown)
- Offer Tier: Audit / Sprint / Transformation (dropdown)
- COI Amount: Dollar value calculated on discovery call (currency field)
- Decision Maker: Name + title of economic buyer (text)
- Champion: Name + title of internal advocate (text)
- Stall Reason: Budget / Authority / Timing / Competition / No Decision (dropdown)
- Source Channel: Website Form / LinkedIn Inbound / LinkedIn Outbound / Referral / Past Client (dropdown)
- Pre-Call Questionnaire Status: Not Sent / Sent / Completed (dropdown)

**3. Views and Filters:**
- "Active Pipeline" view: All deals NOT in Closed Won/Lost
- "Stalled Deals" view: Deals in Proposal Sent or Negotiation for >7 days
- "Past Client Expansion" view: Deals tagged as Source Channel = Past Client
- "This Week's Calls" view: Deals in Discovery Call stage with call date this week

**4. Setup Instructions:**
Step-by-step HubSpot UI navigation for creating each element above. Write as if Saksham has never used HubSpot before:
- How to create the pipeline (Settings > Objects > Deals > Pipelines)
- How to add custom properties (Settings > Properties > Deal Properties > Create Property)
- How to create saved views
- How to connect Calendly (if applicable on free tier)

**5. Integration Notes:**
- /solutions pre-qualification form (WEB-07): Currently logs to console. Note that connecting this to HubSpot requires either HubSpot Forms embed or a custom API integration. Recommend HubSpot Forms embed as simplest approach (replace the current form with HubSpot's embedded form).
- Calendly: HubSpot-Calendly integration auto-creates deals when calls are booked (available on HubSpot Free + Calendly Pro)
- Email: Log emails to HubSpot by BCC'ing the HubSpot logging address

Note: This is a Free tier configuration guide. Flag any features that require Sales Hub Starter ($20/mo) as optional upgrades.
