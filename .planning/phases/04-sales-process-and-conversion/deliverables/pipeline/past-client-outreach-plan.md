### Follow-Up Rules

1. **No response after first message:** Wait 7 days. Send one follow-up referencing a new insight, industry development, or relevant case study result. Do NOT send "just checking in."
2. **No response after second attempt:** Add to 90-day check-in list. Continue engaging with their content on LinkedIn (likes, comments) to stay visible.
3. **Response but not ready:** Log the timing objection. Add to 30-day check-in list. Keep them warm through content engagement.
4. **Response with interest:** Move immediately to booking a discovery call. Do not over-explain in messages. Use Calendly link.

### Follow-Up Message Template (for Day 7 after no response)

> Hi [Name],
>
> Quick follow-up -- saw [relevant industry article / new development / recent case study result] and thought of [Company].
>
> [One sentence connecting the insight to their specific situation.]
>
> No rush -- if the timing is right, happy to chat.

## 5. Tracking

### HubSpot Logging

All past client outreach is logged in HubSpot, separate from cold pipeline activity:

- **Source Channel:** "Past Client" (custom property)
- **ICP Segment:** "Past Client - Expansion" tag
- **Deal Stage:** Start at "Lead" when first message is sent; advance as the conversation progresses
- **Custom Notes:** Reference the original case study and the specific expansion opportunity proposed

### Tracking Fields Per Client

| Field | Description |
|-------|-------------|
| First Message Sent | Date the re-engagement message was sent |
| Channel | LinkedIn DM or Email |
| Response Received | Date of first response (or "No Response") |
| Response Sentiment | Positive / Neutral / Negative / No Response |
| Call Booked | Date of discovery call (if booked) |
| Expansion Opportunity | The specific next engagement proposed |
| Estimated Deal Size | Based on expansion opportunity scope |
| Outcome | In Progress / Call Booked / Not Interested / Timing Later / Closed Won |
| Next Follow-Up Date | When to check in next |

### Weekly Review

Every Friday, review the past client outreach tracker:
- How many messages sent this week?
- How many responses received?
- How many calls booked?
- Which clients need follow-up next week?
- Any clients to move between tiers based on new information?
