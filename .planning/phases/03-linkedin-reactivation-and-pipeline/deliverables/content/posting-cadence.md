# LinkedIn Daily Posting Cadence

> Post once per day, 7 days/week. No 2x/day posting (40% reach drop per post when double-posting). This cadence starts Day 8+ after the 7-day warmup sequence.

## Format Assignments

| Format | When to Use | Specs |
|--------|------------|-------|
| Text-only | Builder logs, explainers, contrarian takes, quick insights, personal stories | 150-250 words. Strong hook in first 2 lines. Short paragraphs (1-2 sentences). Line breaks between sections. |
| Carousel (PDF) | Case study breakdowns, tool comparisons, frameworks, data-heavy content | 6-10 slides. 1080x1350px. Dark background (#0A0A0B) with white text and Electric Mint (#00E5A0) accents. Slide 1 = hook headline. Last slide = takeaway + optional CTA. |

**Carousel production:** Use Canva Pro with 5 master templates (case study, framework, comparison, builder log, data insight). Dark + mint brand. Export as PDF for document posts. Target: 15 min per carousel once templates are set.

## CTA Strategy

### Rules
- **3 CTA posts per week maximum.** Tuesday, Thursday, Saturday.
- **DM-based CTAs only.** No "comment [keyword]" -- algorithm penalizes engagement bait in 2026.
- **No links in post body.** -60% reach penalty. If linking, put in first comment (still -15-20% penalty, use sparingly).
- **Match CTA to content.** Case study post -> offer related audit. Explainer post -> offer deeper resource. Tool comparison -> offer comparison spreadsheet.
- **4 non-CTA posts per week.** Let the content build authority without always asking for something.

### CTA Rotation

| CTA Keyword | Use For | Frequency |
|-------------|---------|-----------|
| "DM me 'audit'" | After case study results, authority content | 1-2x/week |
| "DM me 'systems'" | After framework or deep-dive content | 1x/week |
| "Follow for daily AI systems content" | Soft CTA on personal/trending posts | 1x/week (Friday) |

### What NOT to Do
- Never use "Comment '[keyword]'" -- LinkedIn 2026 NLP flags this as engagement bait
- Never use hashtags -- 3+ hashtags = 70% lower reach
- Never pitch in the post body -- the DM is where the conversation starts
- Never use the same CTA phrasing 7 days in a row -- rotate between audit, systems, and soft follow

## Daily Engagement Protocol

### Pre-Posting Engagement (Non-Negotiable)
1. **15-20 minutes BEFORE posting your content each day**
2. Like and comment on 15-20 posts from:
   - Target ICP accounts (PE partners, startup founders/CTOs)
   - AI thought leaders (builds association)
   - People who engaged with your posts yesterday
3. Comments must add genuine value -- never "Great post!" or emoji-only
4. Comment format: Share a specific insight, ask a follow-up question, or add a data point

### Post-Publishing Engagement
1. **Reply to every comment within 15 minutes** -- 90% algorithmic boost
2. DM anyone who leaves a thoughtful comment and matches ICP (within 24 hours)
3. Monitor profile views -- if someone views your profile after engaging, they are warm prospects

### Time Budget
| Activity | Time | When |
|----------|------|------|
| Pre-post engagement | 15-20 min | Before posting (morning) |
| Reply to comments | 10-15 min | Within 15 min of posting + check at lunch + evening |
| DM warm engagers | 10-15 min | Afternoon batch |
| **Total daily** | **35-50 min** | Spread across the day |

## Weekly Batch Production Workflow

### Sunday: Draft the Week (2-3 hours)

1. **Review trending topics** (15 min)
   - Check what AI topics blew up this week (Claude Code updates, AI agent news, MCP developments, vibe coding stories)
   - Adjust Monday-Wednesday-Friday-Saturday posts to ride any trending wave
   - Thursday and Sunday (authority) stay as planned unless a case study is time-sensitive

2. **Draft all 7 posts** (90 min)
   - Use the templates from post-templates.md as structure guides
   - Pull from the content bank (Week2Plus_Content_Bank.md) for validated ideas
   - Pull case study metrics from real deployments for Thursday authority posts
   - Write hooks first, then fill in body

3. **Quality check** (15 min)
   - Run each draft against the voice checklist: operator-first, no banned words, no hashtags, specific metrics over vague claims
   - Verify word count (150-250 per post)
   - Check for topic diversity -- no single topic should appear more than twice in a week

4. **Schedule or queue** (15 min)
   - Use Taplio or Buffer to schedule Mon-Sat
   - Keep Sunday post manual (contrarian/process posts benefit from real-time engagement)

### Mid-Week Adjustments
- If a trending topic breaks mid-week (new AI model, major Claude Code update, viral vibe coding story), swap the next trending slot for a reaction post
- Authority posts (Thursday, Sunday) stay locked -- these are planned content, not reactive
- If a post goes viral (10x normal engagement), post a follow-up or part 2 within 48 hours -- replace the next trending slot

## Posting Timing

- **Post between 8:00-10:00 AM IST** (aligns with morning LinkedIn scroll in India + evening scroll for US East Coast)
- **Minimum 18-24 hours between posts** -- high-performing posts generate engagement for 48-72 hours; let each post breathe
- **Reply to first comments within 15 minutes** -- this is the most important window for algorithmic distribution
