# LinkedIn Banner Design Brief

*Specification for Canva creation. This is a brief -- not the banner itself.*

---

## Dimensions

**1584 x 396 px** (LinkedIn recommended banner size)

---

## Design Direction

**Social proof / metrics showcase on dark background.**

This is NOT a title banner ("AI Systems Architect" in big text). The banner communicates credibility through numbers at a glance. A profile visitor should absorb the key metrics within 3-6 seconds without reading the About section.

---

## Layout

### Structure: Left-aligned text block + right-side metric cards

```
|----------------------------------------------------------------------|
|                                                                      |
|  [Left 60%]                           [Right 40%]                    |
|                                                                      |
|  "Production AI Systems               [ 14 ]   [ $2.4M ]  [ 0 ]    |
|   That Deliver Results"                deploys   saved     failures  |
|                                                                      |
|  sakshamsolanki.com                                                  |
|                                                                      |
|----------------------------------------------------------------------|
```

### Left Side (60% width)

- **Primary text:** "Production AI Systems That Deliver Results"
  - Font: Bold sans-serif (Satoshi or Montserrat if Satoshi unavailable in Canva)
  - Size: Large enough to read at banner scale (~28-32px equivalent)
  - Color: White (#FAFAFA)
  - Line height: Tight

- **Website URL:** sakshamsolanki.com
  - Font: Regular weight, smaller than primary text
  - Color: Electric Mint (#00E5A0)
  - Positioned below primary text with spacing

### Right Side (40% width)

Three metric cards in a horizontal row:

| Metric | Display | Label |
|--------|---------|-------|
| Deployments | **14** | production deployments |
| Savings | **$2.4M** | in client savings |
| Failures | **0** | failed deployments |

- **Numbers:** Large, bold, Electric Mint (#00E5A0)
- **Labels:** Small, white or light gray (#E5E5E7), below each number
- **Card style:** No visible card borders -- just the numbers and labels floating on the dark background. Alternatively, very subtle dark gray (#1C1C1F) card backgrounds with 1px border (#2A2A2E) if the numbers need visual separation.

## Typography Direction

- **Primary headline:** Bold sans-serif, tight letter-spacing (-0.02em feel)
- **Metric numbers:** Extra bold or black weight -- these are the visual anchor
- **Metric labels:** Regular weight, small size
- **URL:** Regular weight, medium size

Use Satoshi if available in Canva. Fallback: Montserrat, DM Sans, or Poppins (all available in Canva and close to the Satoshi aesthetic).

---

## Visual Rules

1. **No gradients.** Flat, solid dark background only.
2. **No stock imagery.** No robots, no circuit boards, no abstract AI graphics.
3. **No logos** other than the website URL.
4. **Minimal decoration.** A subtle thin horizontal line or small geometric accent in mint is acceptable. No heavy design elements.
5. **Breathing room.** Generous padding on all sides -- the banner should feel clean and premium, not cramped.
6. **Safe zones.** LinkedIn overlays the profile photo on the bottom-left of the banner on desktop. Keep critical content away from the bottom-left quadrant (roughly 200px from left, 150px from bottom).

---

## Mobile Considerations

- LinkedIn crops the banner differently on mobile (center-focused crop)
- Keep all critical elements (metrics, primary text) in the center-right area
- Test the design at both desktop and mobile preview sizes before finalizing

---

## Reference

- **Website aesthetic match:** sakshamsolanki.com uses #0A0A0B backgrounds with #00E5A0 (Electric Mint) accents
- **Impression goal:** "This person has real results, not just a title"
- **Competitor differentiation:** Most AI consultants use title-based banners ("AI Strategy Consultant" in bold). The metrics approach immediately separates Saksham from that pattern.
